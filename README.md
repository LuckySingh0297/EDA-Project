# 🎓 University Data Analysis (EDA Project)

## 📌 Objective

The goal of this project is to perform Exploratory Data Analysis (EDA) on university data to understand key factors affecting student outcomes such as graduation rate, acceptance rate, and overall academic performance.

---

## 📂 Dataset

The dataset contains information about various universities, including:

* Graduation Rate
* Acceptance Rate
* Tuition Fees
* Enrollment
* State-wise distribution

---

## 🔍 Analysis Performed

### 1. Distribution Analysis

* Examined the distribution of graduation rates using histogram and density plots
* Identified that most universities have high graduation rates

### 2. Outlier Detection

* Used boxplots to identify universities with unusually low or high graduation rates
* Found a few outliers indicating performance variation

### 3. Geographic Analysis

* Analyzed number of universities across different states
* Observed uneven distribution of educational institutions

### 4. Relationship Analysis

* Used scatter plots to study relationship between acceptance rate and graduation rate
* Found that more selective universities tend to have better graduation outcomes

### 5. Correlation Analysis

* Used heatmap to identify relationships between different variables
* Identified key factors influencing graduation rate

---

## 📊 Key Insights

* Most universities have graduation rates between **70%–90%**, indicating strong academic performance
* Selective universities (low acceptance rate) generally have higher graduation rates
* Significant variation exists in acceptance rates across institutions
* Some features show strong correlation with graduation outcomes

---

## 💼 Business Implications

* Students aiming for better outcomes should consider more selective universities
* Institutions can analyze factors influencing graduation rate to improve performance
* Policymakers can identify regions with fewer universities and improve accessibility
